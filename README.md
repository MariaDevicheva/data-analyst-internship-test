# Блок 3. ETL Pipeline и аналитическая обработка данных

## Описание

Проект представляет собой ETL-конвейер для извлечения, очистки, преобразования и загрузки данных из нескольких разнородных источников.

Обработка данных завершается загрузкой в аналитическую базу данных SQLite и выполнением аналитических SQL-запросов.

## Выполненные задачи

* извлечение данных из CSV, JSON, XLSX и XML;
* очистка и валидация данных;
* контроль качества данных;
* логирование результатов обработки;
* загрузка данных в аналитическую базу данных;
* выполнение аналитических SQL-запросов;
* контейнеризация приложения с использованием Docker.

## Используемые технологии

* Python
* Pandas
* SQLAlchemy
* SQLite
* Docker
