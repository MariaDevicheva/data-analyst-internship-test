# Аналитический отчёт по данным музыкального стримингового сервиса

## Цель анализа

Целью исследования является анализ пользовательской активности, популярности музыкального контента и региональных предпочтений аудитории на основе данных музыкального стримингового сервиса.

Для анализа были использованы данные об исполнителях, альбомах, песнях, жанрах и истории прослушиваний пользователей.

# 1. Самый популярный Rock-альбом

## Результат

| Альбом        | Исполнитель     | Прослушивания |
| ------------- | --------------- | ------------- |
| Night Visions | Imagine Dragons | 345           |

## Вывод

Альбом **Night Visions** группы **Imagine Dragons** является наиболее популярным Rock-альбомом в исследуемом датасете.

Количество прослушиваний существенно превышает показатели большинства других релизов жанра Rock.

## Возможные причины

* высокая узнаваемость исполнителя;
* наличие нескольких популярных хитов внутри альбома;
* длительный жизненный цикл контента.

## Рекомендации

* использовать композиции альбома в рекомендательных подборках;
* продвигать похожих исполнителей аудитории Rock-сегмента;
* анализировать характеристики наиболее популярных треков альбома.

## Результат

| Исполнитель  | Количество песен в топ-20% |
| ------------ | -------------------------- |
| Depeche Mode | 5                          |

## Вывод

Исполнитель **Depeche Mode** имеет максимальное количество композиций среди самых прослушиваемых треков платформы.

Это свидетельствует о высокой вовлечённости аудитории и устойчивом интересе к каталогу исполнителя.

Пользователи активно потребляют не отдельный хит, а сразу несколько композиций артиста.

Подобное поведение является индикатором сильного бренда исполнителя внутри музыкального сервиса.

## Рекомендации

* использовать исполнителя в промо-кампаниях;
* формировать персональные рекомендации на основе его аудитории;
* анализировать схожих исполнителей для расширения пользовательских сценариев прослушивания.

# 3. Альбом с наибольшим количеством коллабораций

## Результат

| Альбом    | Основной исполнитель | Количество коллабораций |
| --------- | -------------------- | ----------------------- |
| Take Care | Drake                | 4                       |

## Вывод

Альбом **Take Care** исполнителя **Drake** содержит наибольшее количество совместных композиций среди всех альбомов в выборке.

## Бизнес-интерпретация

Коллаборации позволяют объединять аудитории нескольких исполнителей и увеличивать охват контента.

Высокое количество совместных треков может положительно влиять на популярность релиза и вовлечённость пользователей.

## Рекомендации

* анализировать эффективность коллабораций как инструмента роста прослушиваний;
* использовать данные о совместных исполнениях для построения музыкального графа связей артистов;
* учитывать коллаборации в рекомендательных алгоритмах.

* музыкальные предпочтения пользователей не подвержены сильной сезонности;
* сервис поддерживает стабильный уровень вовлечённости.

## Рекомендации
